Weld Extensions
===============

This is a set of portable extensions for Contexts and Dependency Injection (CDI) for
Java EE (JSR-299) consisting of:

* Trial improvements to the CDI API 
* Utilities for writing frameworks and libraries based on CDI
* Extensions allowing for easy configuration of applications. 

This distribution, as a whole, is licensed under the terms of the Apache Public
License (see apl.txt).

Documentation can be found at
http://seamframework.org/WeldExtensions

This distribution consists of:

artifacts/
   -- Weld Extension binaries, sources, and javadocs packaged as jars

doc/
   -- Reference guide for Weld Extensions

lib/
   -- Libraries dependencies for running Weld Extensions
